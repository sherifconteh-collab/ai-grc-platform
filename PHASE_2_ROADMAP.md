## Phase 3 — Upcoming

### Framework Coverage Expansion
- **PCI DSS 4.0** — Payment Card Industry Data Security Standard
- **CIS Controls v8** — Center for Internet Security critical security controls
- **COBIT 2019** — Governance framework for enterprise IT
- **FedRAMP** — Federal Risk and Authorization Management Program

### Desktop & Deployment
- macOS Universal binary (combines arm64 + x64 into a single .dmg via `@electron/universal`)
- Improved auto-updater for multi-arch macOS builds

### AI & Intelligence
- SBOM integration for AI model supply-chain risk (AI-SBOM / AIBOM)
- Enhanced multi-agent orchestration with persistent memory
- AI model performance benchmarking within governance workflows

### Enterprise & Scale
- Custom framework builder (define your own controls and crosswalks)
- Advanced analytics and reporting (trend analysis, executive summaries)
- Multi-tenant improvements (org-level branding, custom domains)
- Continuous monitoring integrations (Wiz, Orca, AWS Security Hub)
- SSO improvements (SCIM provisioning, group-to-role mapping)

### Community
- Additional crosswalk mappings for Phase 3 frameworks
- GDPR ↔ CCPA ↔ ISO 27701 privacy crosswalks
- FedRAMP ↔ NIST 800-53 mapping completion
